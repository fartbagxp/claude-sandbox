# Directory Restructure Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Move all build artifacts and runtime state into the repo directory, decouple the hostname allowlist from `~/.claude/settings.json`, and make the project self-contained for new users.

**Architecture:** Replace `~/.claude-sandbox/` with repo-local `build/` and `run/` directories. Move `config.sh` into `config/` alongside a new plain-text `allowlist.txt`. Update all scripts to derive paths from the repo root via `SCRIPT_DIR`.

**Tech Stack:** Bash, dnsmasq, nftables, cloud-hypervisor, virt-builder

## Task 4: Rewrite `generate_dnsmasq_nftset_conf` to read plain-text allowlist

**Files:**
- Modify: `claude-sandbox.sh:122-138` (`generate_dnsmasq_nftset_conf` function)

**Step 1: Replace the function**

Replace the entire `generate_dnsmasq_nftset_conf` function. Old version uses `jq` to parse JSON from `CLAUDE_SETTINGS`. New version reads plain text from `ALLOWLIST_FILE`.

Replace:

```bash
generate_dnsmasq_nftset_conf() {
    echo "  Generating dnsmasq nftset config from ${CLAUDE_SETTINGS}..."
    if [ ! -f "$CLAUDE_SETTINGS" ]; then
        echo "ERROR: Claude settings not found at ${CLAUDE_SETTINGS}" >&2
        exit 1
    fi

    jq -r '.sandbox.network.allowedHosts[]
        | select(startswith("_comment") | not)' "$CLAUDE_SETTINGS" \
        | while read -r host; do
            echo "nftset=/${host}/4#inet#claude_filter#allowed_ips"
        done > "$DNSMASQ_NFTSET_CONF"

    local count
    count=$(wc -l < "$DNSMASQ_NFTSET_CONF")
    echo "  Generated ${count} nftset entries."
}
```

With:

```bash
generate_dnsmasq_nftset_conf() {
    echo "  Generating dnsmasq nftset config from ${ALLOWLIST_FILE}..."
    if [ ! -f "$ALLOWLIST_FILE" ]; then
        echo "ERROR: Allowlist not found at ${ALLOWLIST_FILE}" >&2
        echo "  Copy the example: cp config/allowlist.txt.example config/allowlist.txt" >&2
        exit 1
    fi

    # Strip comments and blank lines, generate dnsmasq nftset directives
    grep -v '^\s*#' "$ALLOWLIST_FILE" | grep -v '^\s*$' \
        | while read -r host; do
            echo "nftset=/${host}/4#inet#claude_filter#allowed_ips"
        done > "$DNSMASQ_NFTSET_CONF"

    local count
    count=$(wc -l < "$DNSMASQ_NFTSET_CONF")
    echo "  Generated ${count} nftset entries."
}
```

**Step 2: Commit**

```bash

git commit -m "refactor: read allowlist from plain-text file instead of settings.json"
```
